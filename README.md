# 🎓 Facial Recognition Based Attendance System for Teachers and Students Using Machine Learning

An intelligent attendance management system that automates student attendance using **Face Recognition** and **Machine Learning**. The system captures facial images through a webcam, recognizes registered users, and marks attendance automatically, reducing manual effort and preventing proxy attendance.

---

## 📌 Overview

Traditional attendance systems are time-consuming and prone to errors. This project provides an automated solution that identifies students and teachers using facial recognition technology and records attendance accurately.

The application utilizes **OpenCV** for face detection and recognition, along with machine learning techniques to train and identify registered faces in real time. :contentReference[oaicite:0]{index=0}

## ⚙️ How It Works

1. Register a new student or teacher.
2. Capture multiple facial images.
3. Train the facial recognition model.
4. Start the attendance system.
5. The webcam detects faces in real time.
6. Attendance is automatically marked and stored.

## 🚀 Installation
